Humble is a simple approach to environment variables in Scratch. You simply register them by setting their value, and you're done. Resolve them by putting strings in a single reporter, and now you have an environment variable system.

## Key Features

- **Mint-based:** Humble uses the Mint development toolchain, so it'll be rock-hard and maintainable in the future.
- **Easy to use:** Simply register by setting a variable's name, and you're done. No busy work!
